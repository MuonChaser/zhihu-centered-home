# 知乎 · 简净居中

一个 Tampermonkey / Violentmonkey 用户脚本：把知乎网页版首页、普通问题页和回答详情页的内容置于页面中央，隐藏右侧栏与无关推广框，并把顶栏精简为知乎 Logo 与居中的搜索框。问题页会保留问题标题、话题、描述与必要操作。搜索页、文章页和个人主页不受影响。

## 安装

1. 在 Firefox 中安装 Tampermonkey 或 Violentmonkey。
2. 点击 **[自动安装 / 更新“知乎 · 简净居中”](https://github.com/MuonChaser/zhihu-centered-home/raw/refs/heads/main/zhihu-centered-home.user.js)**。
3. 在脚本管理器的安装页确认安装，然后刷新知乎首页、普通问题页或回答详情页。

已经安装旧版本时，再次打开安装链接即可更新到最新版。

## 效果

- 首页信息流、普通问题页和回答详情页正文保持知乎原生主栏的 694px 宽度并水平居中。
- 顶栏使用始终一致的半透明磨砂背景与轻阴影，隐藏频道导航、直答、创作、消息、私信和头像，只保留知乎 Logo 与严格居中的 560px 搜索框。
- 顶栏样式不随滚动位置切换，避免滚动或返回顶部时出现视觉跳变。
- 隐藏搜索框中由知乎动态填入的推荐占位文字，输入区域默认保持空白。
- 隐藏首页推广横幅与发布卡片、问题页右侧栏以及滚动时重复出现的问题标题栏。
- 问题标题、话题标签、问题描述与必要操作保留，并整理成与回答正文同宽的圆角卡片。
- 窄屏（小于 1000px）不改变知乎原有的响应式布局。

## 维护

知乎可能调整网页的 CSS 类名。如果右侧栏重新出现，请在 issue 中附上浏览器版本、页面链接和截图（请打码个人信息）。
